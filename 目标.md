# 第七次迭代

## 目标

使用外置文件设定敌人的出场模式

BOSS战，所有敌机不继续刷新

## 文件格式

采用json格式：

```json
{
    "enemies" : [
        {
            "className" : "敌人1的类名",
            "appearBy" : "出场依据：timeInterval——根据时间间隔",
            "appearMode" : "出场位置：random——随机位置|fix——固定位置",
            "mu" : "时间间隔的均值，仅在time依据下使用",
            "std" : "时间间隔的标准差，仅在time依据下使用",
            "posX" : "出场位置的X轴，仅在fix位置下使用"
        },
        {
            "className" : "敌人2的类名",
            ......
        }
    ]
}
```
